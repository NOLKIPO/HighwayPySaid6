# Highway Signer — Управление обменом файлами

## Описание
Highway Signer — это приложение для управления SFTP-туннелями, предназначенное для обмена файлами между клиентом и процессором с поддержкой различных режимов работы. Приложение предоставляет удобный графический интерфейс для настройки и мониторинга рабочих мест.

## Особенности
- Поддержка нескольких рабочих мест
- Различные режимы работы (клиент, процессор, клиент-подпись, процессор-подпись)
- Автоматическая синхронизация файлов
- Генерация SSH-ключей
- Ведение подробных логов
- Удобный графический интерфейс с цветовой индикацией статусов
- Подсказки к кнопкам и элементам управления
- Современный дизайн с приятной цветовой схемой

## Режимы работы
- **Клиент**: отправляет исходящие файлы и получает входящие
- **Процессор**: отправляет исходящие файлы и получает входящие (в обратном направлении)
- **Клиент-подпись**: специальный режим для обработки подписей
- **Процессор-подпись**: специальный режим для обработки подписей (в обратном направлении)

## Установка и запуск
1. Убедитесь, что у вас установлен Python 3.x и PyQt6
2. Установите зависимости: `pip install paramiko pyqt6`
3. Запустите приложение: `python main_pysaid.py`

- Яркие кнопки с эффектами наведения
